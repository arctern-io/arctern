ST_Distance¶

arctern.wrapper_func.ST_Distance(left, right)[source]¶

Calculate the distance between two geometries.

For every (left, right) pair with the same offset value in left and right, calculates the minimum 2D Cartesian (planar) distance between left and right.

Parameters

  • left (pyarrow.array.string) -- Geometries organized as WKB.

  • right (pyarrow.array.string) -- Geometries organized as WKB.

Returns

An array of double.

Return type

pyarrow.array.double

Example
>>> import pandas
>>> import arctern
>>> p11 = "LINESTRING(9 0,9 2)"
>>> p12 = "POINT(10 2)"
>>> data1 = pandas.Series([p11, p12])
>>> p21 = "POLYGON((0 0,0 8,8 8,8 0,0 0))"
>>> p22 = "POLYGON((0 0,0 8,8 8,8 0,0 0))"
>>> data2 = pandas.Series([p21, p22])
>>> rst = arctern.ST_Distance(data2, data1)
>>> print(rst)
    0    1.0
    1    2.0
    dtype: float64
