ST_Overlaps¶

arctern_pyspark._wrapper_func.ST_Overlaps(left, right)[source]¶

Check whether geometries "spatially overlap".

For every (left, right) pair with the same offset value in left and right, check if they "spatially overlap". "Spatially overlap" here means two geometries intersect but one does not completely contain another.

Parameters

  • left (pyarrow.array.string) -- Geometries organized as WKB.

  • right (pyarrow.array.string) -- Geometries organized as WKB.

Returns

An array of booleans.

Return type

pyarrow.array.boolean

Example
>>> from pyspark.sql import SparkSession
>>> from arctern_pyspark import register_funcs
>>> spark_session = SparkSession           .builder           .appName("Python Arrow-in-Spark example")       >>> register_funcs(spark_session)
>>> test_data = []
>>> test_data.extend([('POLYGON((1 1, 4 1, 4 5, 1 5, 1 1))', 'POLYGON((3 2, 6 2, 6 6, 3 6, 3 2))')])
>>> test_data.extend([('POINT(1 0.5)', 'LINESTRING(1 0, 1 1, 3 5)')])
>>> overlaps_df = spark_session.createDataFrame(data=test_data, schema=["left", "right"]).cache()
>>> overlaps_df.createOrReplaceTempView("overlaps")
>>> spark.sql("select ST_Overlaps(ST_GeomFromText(left), ST_GeomFromText(right)) from overlaps").show()
+----------------------------------------------------------+
|ST_Overlaps(ST_GeomFromText(left), ST_GeomFromText(right))|
+----------------------------------------------------------+
|true                                                      |
+----------------------------------------------------------+
|false                                                     |
+----------------------------------------------------------+
