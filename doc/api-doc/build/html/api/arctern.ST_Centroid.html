ST_Centroid¶

arctern.ST_Centroid(geos)[source]¶

Compute the centroid of geometry.

For every geometry in geometries, compute the controid point of geometry.

Parameters

geos (pyarrow.array.string) -- Geometries organized as WKB.

Returns

Geometries organized as WKB.

Return type

pyarrow.array.string

Example
>>> import pandas
>>> import arctern
>>> data = ["POLYGON((0 0,1 0,1 1,0 1,0 0))", "POLYGON((0 0,0 8,8 8,8 0,0 0))"]
>>> data = pandas.Series(data)
>>> rst = arctern.ST_AsText(arctern.ST_Centroid(data))
>>> print(rst)
    0    POINT (0.5 0.5)
    1    POINT (4 4)
    dtype: object
